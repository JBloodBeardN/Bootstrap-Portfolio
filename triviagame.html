Trivia Game

ReadMe

Trivia Game is a UT coding bootcamp homework activity. It utilizes timing events to all a User to race the clock and submit a Trivia Game. The game keeps track of time allowed for each question, and the total number of questions answered correctly. I am currently using questions inspired by the GateKeeper in Monte Python's Holy Grail. The correct 'answers' to the questions are inspired by the events that transpired in the movie.

The game uses form components controlled by a show/hide mechanic which is slave to timing events. Each question answered is immediately evaluated and result of evaluation is presented to the user. This updates javascript variables for number of questions asked and answered correctly. At the end of the game, the user is presented a button to clear the values of the javascript variables in order to reset the game.
