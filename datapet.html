Dat-A-Pet

ReadMe

The Dat-A-Pet project was a collaborative effort with 3 other members of the coding bootcamp I attended. Dat-A-Pet was first created for 'Project 2', as a Node.js, Express Server, HandleBars.js/Bootstrap, MySQL/Sequelize stack. The project scope was found to be a loftier goal than achievable and resulted in a homepage, user creation, pet object creation, minimal user dashboard view, and json output for admin API calls.

Dat-A-Pet was revised for 'Project 3' however much of the work had to be redone, and most all of the project had to be refactored for new project requirements. Changes to the stack to now include React javascript library instead of HandleBars.js and a switch from Bootstrap framework to Materialize(similar css library). An update to database design to support a many-to-many relationship between pet owners and pets also cascaded to refactor all API calls. The Dat-A-Pet team was able to deliver MVP functionality: [Create a User, Create Pet Objects, Share Pet Object with another user]. Functionality can be tested using the seed account TEST@email.com, password: TEST. This user has default pets created which can be shared with your user, and vice-versa, your pets can be shared with that user. The shared user only has view permissions for pets that did not originate with them. HINT: Logout button and Share my Pet functions are within the side nav bar of the dashboard page, accessible through the header button 'Nav'
