Giphy API

ReadMe

Giphy API is a homework activity of the UT coding bootcamp. It is an exercise in utilizing an API and handling the response data using array methods. The API is hit using values attached to buttons in the header. This comprises the API query string. The response data is then handled within a loop and bootstrap components are added for each response item.

A user is able to add a button to the header, which then becomes available to key the API service. I am attempting to filter the GIFs returned by the services internal rating system, which is currently scoped to 'G' rated GIFs. NOTICE: I have no control over incorrectly rated GIFs, nor am I controlling for user input against a profanity list. Use with your own discretion. ADMISSION: I mistakenly call the service Gliphy as you can see in my file names.
